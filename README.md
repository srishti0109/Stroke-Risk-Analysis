# 🧠 Stroke Risk Analysis Dashboard

## 📌 Project Overview

This project analyzes a healthcare dataset to identify the key factors associated with stroke risk. Data cleaning and preprocessing were performed using Python (Pandas), and an interactive dashboard was created in Power BI to present meaningful insights.

---

## 🎯 Objectives

- Analyze stroke risk across different age groups.
- Compare stroke rates based on hypertension and heart disease.
- Analyze average glucose levels and patient age by stroke status.
- Study the impact of smoking habits on stroke risk.
- Build an interactive dashboard for data-driven insights.

## 📈 Key Insights

- Stroke risk increases significantly with age.
- Patients with heart disease have the highest stroke rate.
- Hypertension is associated with a higher stroke rate.
- Stroke patients have a higher average glucose level.
- Former smokers show the highest stroke rate among all smoking categories.
